# Hyphens (One-Handed Weapons)
Compound modifiers describing a noun get hyphens: 
> This is a two-handed sword.
> That one-handed weapon is legal.
> This sword is two-handed.

_(Note: That only applies to adjectives. "wielded with two hands" does not get a hyphen.)_
## Capitalization of Hyphenated Words
- **Sentence start:** capitalize the first word, which for a hyphenated compound means only the first element
	- _Two-handed swords are expensive._
- **Title case:** capitalize both elements
	- _Two-Handed Swords Throughout History._
- **Mid-sentence:** all lowercase
	- _Those two-handed swords are heavy._

Note: Tags use the title case
# Longsword vs Long Sword
We have long swords, not longswords. Reasoning:
- "Longsword" is the term for a specific style of sword. A katana is not a longsword, but it is a long sword. 
- We have long swords, short swords, and two-handed swords. It is not possible to have longswords, shortswords, and two-handedswords.
